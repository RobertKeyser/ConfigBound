# Bind

A Bind is the mechanism that retrieves the values of the [Element](../element.mdx) from their source. There are multiple types of Binds, each with their own specific way of retrieving the values.

- [EnvVarBind](./binds/envVar.mdx) - Retrieves the values from the environment variables.
